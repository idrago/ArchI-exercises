Scrivere una procedura RISC-V per il calcolo del massimo comune divisore di due numeri interi positivi \(a\) e \(b\). A tale scopo, implementare l'algoritmo di Euclide come procedura \(MCD(a,b)\) da richiamare nel main. L'algoritmo di Euclide in pseudo-codice Ã¨ il seguente: 

int MCD(int a, int b) {
    while (a != b) { 
        if (a > b)
            a = a - b;
        else
            b = b - a;
    }
    return a;
}
Il valore di ritorno deve essere lasciato sul registro \(a0\).
Attenzione:
  • Incollare nel campo di sotto soltanto il contenuto della funzione \(MCD\) (in RISC-V)
  • Attenzione alle convenzioni di chiamata!
