Scrivere il codice per invertire un array di double word in memoria (size elementi).


Codice C equivalente: 
for (int i = 0; i < size/2; i++)  {
    long temp = array[i];
    array[i] = array[size-i-1];
    array[size-i-1] = temp;
}
