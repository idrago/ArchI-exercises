Scrivere il bubble sort in RISC-V per ordinare un array di double word in memoria (size elementi).

Codice C equivalente: 
int change = 1;
while (change) {
    change = 0;
    for (int i = 0; i < size-1; i++)  {
        if (array[i] > array[i+1]) {
            long temp = array[i];
            array[i] = array[i+1];
            array[i+1] = temp;
            chance = 1;
        }
    }
}
