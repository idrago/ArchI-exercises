Scrivere il codice RISC-V che verifichi se un array di size elementi in memoria (double word contigue) contiene una sequenza ordinata di numeri interi (crescente). Il valore ottenuto (0 se non è ordinato, 1 se è ordinato) va salvato in una variabile identificata con l'etichetta result.

Codice C equivalente: 
result = 1;
for (int i = 0; i < size-1; i++) {
    if (array[i] > array[i+1]) {
        result = 0; 
        break;
    }
}
Attenzione:
  • Usare il seguente codice main per lo sviluppo e il debugging nel simulatore RARS
  • Completare il codice nel campo sottostante
.globl _start
.data
    size:   .word  8                  # size of the array        
    array:  .dword 1,2,3,4,5,6,7,8    # array of long integers
    result: .word  -1                 # store the result
    
.text

    # completare con il codice assembly RISC-V nel campo sottostante
