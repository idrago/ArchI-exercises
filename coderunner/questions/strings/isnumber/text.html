Scrivere una funzione RISC-V \(isnumber\) che controlli se una stringa ricevuta come parametro è la rappresentazione di un numero intero positivo in ASCII. Controllare vuol dire: restituire 1 se la condizione è vera, 0 altrimenti. \(isnumber\) deve utilizzare la funzione \(digit\) realizzata nell'esercizio precedente.

Il valore di ritorno deve essere lasciato sul registro \(a0\).

Il seguente codice in \(C\) realizza \(isnumber\) (convertilo in RISC-V): 
int isnumber(char *numstr) {
    for (int i = 0; numstr[i] != '\0'; i++);
        if (!digit(numstr[i]))
            return 0;
    return 1;
}
